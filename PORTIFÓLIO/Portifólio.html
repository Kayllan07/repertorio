PROJETOS

Jogo da Velha / HTML x CSS

Este projeto é uma versão interativa do tradicional Jogo da Velha, desenvolvido com HTML, CSS e JavaScript. O objetivo do jogo é alinhar três símbolos iguais em uma linha, coluna ou diagonal antes do adversário. Os jogadores representam duas tecnologias web populares: HTML e CSS.
Conclusão
Este Jogo da Velha oferece uma maneira divertida e interativa de jogar um clássico jogo de tabuleiro no navegador. Com um design visualmente atraente e uma jogabilidade simples, ele proporciona uma experiência agradável para os jogadores.

Quiz / Gestão Startup

Este é um jogo de quiz interativo sobre "Gestão de Startups" desenvolvido em HTML, CSS e JavaScript. A interface é projetada para ser intuitiva e visualmente atraente, utilizando o framework Bootstrap para estilização adicional.
Conclusão
Este quiz é uma ferramenta educacional divertida e interativa, projetada para testar e melhorar o conhecimento dos usuários sobre gestão de startups. Com uma interface clara e responsiva, ele oferece uma experiência de usuário agradável e envolvente, combinando elementos visuais e auditivos para maximizar a interatividade e o aprendizado.

Jogo da Memória / Robótica

Este projeto consiste em um Jogo da Memória interativo desenvolvido utilizando HTML, CSS e JavaScript. O jogo foi projetado com um tema de robótica, com cartas representando diferentes componentes eletrônicos, como Arduino, LEDs, sensores e muito mais. O objetivo do jogo é encontrar todas as cartas correspondentes em um menor número de tentativas possível.
Conclusão
Este Jogo da Memória de Robótica é uma ótima maneira de aprender e se divertir ao mesmo tempo. Ele combina habilidades de programação front-end com um design interativo e responsivo, proporcionando uma experiência de jogo agradável e educativa.
